# 原生家庭健康度测试

一个帮助用户了解原生家庭影响的在线测试应用。

## 功能特点

### 📊 6大评估维度
- **情感支持** - 家庭成员之间的情感连接和支持程度
- **沟通质量** - 家庭内部的沟通方式和效果
- **边界感** - 个人空间和家庭界限的平衡
- **冲突处理** - 家庭矛盾的解决方式
- **安全感** - 心理和情感上的安全程度
- **成长环境** - 个人发展和自我实现的支持度

### 🎯 测试特色
- 60道精心设计的问题
- 六芒星雷达图可视化展示
- 0-100分综合评分系统
- 针对性的改善建议
- 专业的心理学理论支持

### 📈 评分等级
- **90-100分** - 非常健康的原生家庭
- **75-89分** - 健康的原生家庭
- **60-74分** - 基本健康，有改善空间
- **45-59分** - 存在一些问题，需要关注
- **30-44分** - 问题较多，建议寻求帮助
- **0-29分** - 严重问题，强烈建议专业咨询

## 技术栈

- React 18
- TypeScript
- Vite
- CSS3 (动画和渐变)

## 开发

npm run dev

# 构建生产版本
npm run build

# 预览生产版本
